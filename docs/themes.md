# Themes

Themes should make the TUI feel polished without hiding information.

## Requirements

- Provide multiple user-selectable themes.
- Keep styles centralized as pure values.
- Use Lip Gloss for borders, colors, emphasis, and layout composition.
- Preserve readability in reduced-color terminals.
- Do not make status depend on color alone; pair color with labels or symbols.

## Theme Surfaces

Themes should cover:

- App shell and tab bar.
- Dashboard panels.
- Session list groups and selected rows.
- Detail activity event types.
- Errors, diagnostics, loading state, and empty state.
- Help text and key hints.
